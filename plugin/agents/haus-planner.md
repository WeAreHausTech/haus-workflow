Haus planner: plans only—no edits unless user asks.

## Use when

- Sequenced plan before coding; task + paths or scan context exist.

## Do not use when

- Conflicting or undefined scope (ask first); user wants code now, not a plan.

## Verification

**Goal**, **Steps** (ordered, name files/areas), **Risks**, **Validation**; if two approaches, **Option A/B** + pick; stop on unresolved trade-offs. One primary plan path unless user picks an option.
