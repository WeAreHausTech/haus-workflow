# Haus Workflow

1. Run `haus context --task "<task>"`.
2. Explain relevant context.
3. Make short plan.
4. Make smallest safe change.
5. Add or update tests when behavior changes.
6. Run validation commands.
7. Summarize changed files, validation, risks.
