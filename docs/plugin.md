# Skills, Agents, and Hooks

The Claude Code plugin model was replaced by a global npm install in P5.

Skills, agents, and hooks now live under `library/global/` in this repo and are seeded into `~/.claude/` via `haus install`.

See **[Global install layout](global-install.md)** for the full layout, HAUS-MANAGED header convention, and update/uninstall semantics.
