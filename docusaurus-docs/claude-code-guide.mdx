# Claude Code in Action

Haus Workflow integrates with **Claude Code** through `/haus-workflow`. After `npm install -g @haus-tech/haus-workflow`, every task below is available from chat — the skill runs the CLI and returns structured results.

## Interactive menu (`/haus-workflow`)

Run without arguments to open the task picker. Options render as checkboxes; reply with a number or type the subcommand directly.

<ClaudeChatMockup
  title="Claude Code"
  path="~/Projects/legacy-storefront"
  inputPlaceholder="/haus-workflow"
  ariaLabel="Haus Workflow interactive menu with checkbox options"
  caption="Checkbox menu matches Claude Code skill UI — select a task or type the subcommand."
  messages={[
    { role: 'user', content: '/haus-workflow' },
    {
      role: 'assistant',
      content: [
        {
          type: 'menu',
          title: 'Haus Workflow — choose a task',
          options: [
            {
              command: 'project:init',
              description: 'Add Haus Workflow to this repo (scan → recommend → apply)',
            },
            {
              command: 'project:clone [name]',
              description: 'Clone a repo from the workspace manifest',
            },
            {
              command: 'project:cloneandsetup [name]',
              description: 'Clone + Node version, deps, and .env setup',
            },
            {
              command: 'project:refresh',
              description: 'Refresh .claude/ and regenerate CLAUDE.md imports',
            },
            {
              command: 'project:doctor',

            },
            { command: 'update', description: 'Update npm package, catalog, and ~/.claude/' },
          ],
        },
      ],
    },
  ]}
/>

## `project:init` — add Haus Workflow to a repo

Runs `haus scan` → `haus recommend` → `haus apply --write`. Idempotent — safe to re-run when the stack or catalog changes.

  title="Claude Code"
  path="~/Projects/legacy-storefront"
  inputPlaceholder="/haus-workflow project:init"

  caption="Init writes .claude/ and .haus-workflow/ from catalog recommendations."

    { role: 'user', content: '/haus-workflow project:init' },

              text: 'haus apply --write — .claude/skills/, .claude/agents/, .claude/commands/',
